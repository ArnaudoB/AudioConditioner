\block{\textsc{Introduction \& The Descriptor Model}}{
    \textbf{AudioConditioner} is a high-level multimodal model that we developed to bridge the gap between visual/textual semantics and music generation. We utilize BLIP for image captioning and the CLAP text encoder for semantic embeddings.
    \medskip
    
    The central component is our \textbf{Descriptor Model}, which is the only trainable component in the entire architecture. We keep all other models (BLIP, CLAP, Stable Audio) completely frozen. The Descriptor acts as a multi-task learning bridge; we map the CLAP text embedding through a shared neural backbone into 12 discrete and continuous musical attributes.
    \medskip
    
    We structurally combine the generated attributes within a \textbf{MusicDescriptor} object. We then use this object to programmatically generate the descriptive text prompt fed into Stable Audio (which has a 47-second generation limit). Finally, to ensure semantic alignment, we generate $N$ candidates and re-rank them using cosine similarity against the CLAP text embedding.
}

\block{\textsc{Phase 1: Generative LLM Distillation}}{
    We trained the \textbf{Descriptor Model} using a parallel corpus mapping natural language to the strict \textbf{MusicDescriptor} format. We built this dataset through prompt mixing and teacher LLM distillation:
    \begin{itemize}
        \item \textbf{Data Generation:} We procedurally generated short scene combinations by randomly mixing predefined characters, settings, and events (e.g., \textit{"A lonely astronaut under a sunset discovers an opportunity."}).
        \item \textbf{Distillation:} We prompted an LLM to act as a composer, taking these scenes and outputting the deterministic JSON MusicDescriptor fitting each scene.
        \item \textbf{Optimization:} We trained the model using an AdaptedMusicDescriptorLoss function (Cross-Entropy for categorical attributes and MSE for continuous target regressions).
    \end{itemize}
    \medskip

\block{\textsc{Phase 2: Contextual Novel Fine-Tuning}}{
    To support narrated audiobooks with adaptive background music (ReadMusic), we fine-tuned the model to handle longer contextual evolution:
    \begin{enumerate}
        \item \textbf{Novel Chunking:} We split public domain novels into consecutive paragraphs (up to 80 words).
        \item \textbf{Target Generation:} We used the teacher LLM to assign progressive MusicDescriptors respecting the storyline's emotional flow.
        \item \textbf{Fine-Tuning Outcomes:} The descriptor internalizes literary context; we aim to prevent abrupt changes and enable continuous emotional transitions for long-form reading.
    \end{enumerate}
}
